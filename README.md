# Zijing Wang - Academic Homepage

Static academic homepage prepared for GitHub Pages.

## Publish at `https://ransheng629.github.io/`

1. Create a new public GitHub repository named exactly `ransheng629.github.io`.
2. Upload `index.html`, `styles.css`, and `favicon.svg` from this folder to the repository root.
3. Open the repository's **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then click **Save**.
6. Wait a few minutes and open `https://ransheng629.github.io/`.

The homepage contains no tracking scripts and requires no build tools.
